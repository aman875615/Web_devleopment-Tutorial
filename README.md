##Time Table Project

## 📌 Overview
This project is a simple **HTML Time Table** webpage that displays a weekly class schedule.  
It is created using only **HTML table tags** like `<table>`, `<tr>`, `<th>`, `<td>`, `rowspan`, and `colspan`.

## 🛠️ Technologies Used
